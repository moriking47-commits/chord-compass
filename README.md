# Chord Compass（五度圏指南）

コード進行を **考えながら学ぶ** ためのブラウザツールです。
五度圏を羅針盤（コンパス）に見立て、「次にどのコードへ進むか」を視覚的に探れます。

コード理論を勉強している途中の人が、解説を読みながら、五度圏・コード表・鍵盤・実際の音で
確かめて、自分の進行を組み立てられることを目指しています。

**produced by DJ moriking**

---

## できること

- **8つの切り口ボタン** … 押すと「おすすめの考え方」が切り替わります
  1. 五度圏（メジャー）
  2. 五度圏（マイナー）
  3. メジャーおすすめ
  4. マイナー（Jazzy）
  5. 機能（トニック / サブドミナント / ドミナントの解説つき）
  6. プリセット：定番 / 王道（カノン進行・小室進行 など）
  7. プリセット：EDM
  8. プリセット：Jazzy
- **五度圏表** … 外周＝メジャー、内周＝平行短調、調号つき。今のコードと、使えるコードを色分け
- **コード表** … 各コードに使えるテンション・代理を色分け（緑＝使える / 紫＝借用 / 青枠＝おすすめ）
- **進行スロット** … 選んだコードを並べて、後から差し替え・通し再生
- **鍵盤表示** … 鳴らしているコードの構成音が光る
- **音色切替** … エレピ / オルガン / パッド / シンセリード / ベル

## 使い方

1. キーを選ぶ
2. 切り口ボタンで「今日はどの考え方で探すか」を選ぶ
3. 解説を読みつつ、五度圏や表のコードを押して音で確かめる
4. 気に入ったコードを進行スロットに並べる
5. 「通しで再生」で流れをチェック、気になる枠は押し直して差し替え

## 使い方のスタンス

このツールが示す色やおすすめは、あくまで **定番の流れの「目安」** です。
音楽理論に「唯一の正解」はありません。色がつかないコードを選ぶ
「外し」や転調こそ、曲の個性になることもあります。
**最後は自分の耳で判断してください。**

## 動かし方

`index.html` をブラウザで開くだけで動きます（インストール不要・オフラインでも可）。
GitHub Pages で公開すれば、誰でもブラウザからアクセスできます。

## 技術メモ

- 素の HTML / CSS / JavaScript のみ（フレームワーク不使用）
- 音は Web Audio API による合成（サンプル音源は不使用）
- ピアノ・ギター音色はサンプル音源での追加を検討中
